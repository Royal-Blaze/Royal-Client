# Royal Client

**Slogan:** A Minecraft PvP client built for Nobles

---

## About
Royal Client is a dedicated Minecraft PvP client crafted to deliver smooth performance and refined gameplay. Built with precision for competitive players, it embodies the noble spirit of battle.

---

## Requirements
To run Royal Client, your device must meet the following resolution standards:

- **Desktop:** Minimum display resolution of 1000 pixels (width) × 700 pixels (height)  
- **Smartphone:** Minimum display resolution of 1000 pixels (width) × 700 pixels (height)
